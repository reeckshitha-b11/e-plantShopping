# e-plantShopping 🌿

## Project Overview
e-plantShopping is a simple e-commerce web application for buying plants online. It allows users to browse different plants, view their details, and manage items in a shopping cart.

## Features
- Display a list of plants with images, names, and prices
- Add plants to the shopping cart
- Increase or decrease quantity of items in the cart
- Remove items from the cart
- Automatic calculation of total price
- Responsive and user-friendly UI

## Technologies Used
- React JS
- Redux Toolkit
- HTML5
- CSS3
- JavaScript (ES6)

## Project Structure
- ProductList.jsx → Displays all available plants
- CartItem.jsx → Handles individual cart items
- CartSlice.jsx → Manages cart state using Redux
- App.jsx → Main application component
- App.css → Styling of the application

## How to Run the Project
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000` in your browser
